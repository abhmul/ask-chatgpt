## Background — the bug (verified from ground truth; re-verify, don't blindly trust)
`src/ask_chatgpt/driver.py` → `BrowserSession.wait_for_completion(self, timeout_s: float = 10.0)` (starts at line ~325). Constants: `_POLL_INTERVAL_S = 0.1` (driver.py:44), `_REAL_COMPLETION_STABLE_S = 2.0` (driver.py:45).

The **real/cdp branch** (driver.py ~352–365) decides completion like this:
```
if self.channel in {"real", "cdp"}:
    latest_text = self._latest_assistant_body_text(latest_assistant)
    now = time.monotonic()
    if latest_text and latest_text == last_text:
        if stable_since is None: stable_since = now
    else:
        last_text = latest_text; stable_since = None
    streaming_visible = self._present("streaming_marker")
    completion_visible = completion_present_on_latest or self._present("completion_marker")
    text_stable = stable_since is not None and now - stable_since >= _REAL_COMPLETION_STABLE_S
    if not streaming_visible and (completion_visible or text_stable):
        return latest_assistant
```
**Root cause:** the `or text_stable` term. During a streaming **micro-pause** — the stop button (`streaming_marker`, real selector `button[data-testid="stop-button"]`) momentarily absent AND the body text momentarily unchanged for ≥ 2.0s **mid-stream** — we get `streaming_visible=False`, `completion_visible=False` (the turn is NOT actually done), but `text_stable=True`. So `not False and (False or True)` ⇒ **returns a half-rendered turn ⇒ clipped text**. This is what produced the real-site clip `GAP15_SEEDED M007-GAP15-1F3845_` (the seeded nonce came back truncated). It affects `->text` generally (UC1 CORE), not just bundles.

Selector facts (from `src/ask_chatgpt/selector_maps/real.json`): real `streaming_marker = button[data-testid="stop-button"]`; real `completion_marker = button[data-testid="copy-turn-action-button"]` (the copy-turn-action button that appears when a turn finishes); real `truncation_marker = ""` (empty ⇒ `_present` returns False ⇒ no truncation detection on real); `download_artifact = ""`. ## Goal / required invariant (you choose the mechanism; justify it in your report)

1. **Affordance-gated:** return only when a completion affordance is visible AND the streaming marker is absent. 3. **Combine 1 and 2.** A return on pure `text_stable` with streaming never seen and no affordance ⇒ forbidden.
